┌──────────────────────────────────────────────────────────────────────┐
│ METHOD 1: GitHub Actions (EASIEST - No Command Line!)               │
└──────────────────────────────────────────────────────────────────────┘

Perfect if you don't have terminal access or prefer using the web interface!

1. Go to your repository's Actions tab:
   https://github.com/cpetrula/bolt-ai-group/actions

2. Click "Create GitHub Issues from README" workflow (left sidebar)

3. Click "Run workflow" button (right side)

4. Type "create" in the confirmation field

5. Click the green "Run workflow" button

6. Wait ~60 seconds for completion

7. View your issues at:
   https://github.com/cpetrula/bolt-ai-group/issues

That's it! No tokens, no command line, no local setup needed!

See GITHUB_ACTIONS_GUIDE.md for detailed visual guide.

┌──────────────────────────────────────────────────────────────────────┐
│ METHOD 2: Command Line (Alternative)                                │
└──────────────────────────────────────────────────────────────────────┘

STEP 1: Get a GitHub Personal Access Token

1. Go to: https://github.com/settings/tokens
2. Click "Generate new token (classic)"
3. Give it a name: "Issue Creator"
4. Select scope: ✓ repo (full control of private repositories)
5. Click "Generate token"
6. COPY THE TOKEN (you won't see it again!)

STEP 2: Run the Quick-Start Script

Open your terminal and run:

    export GH_TOKEN=paste_your_token_here
    ./quick-start.sh

That's it! The script will:
  ✓ Install PyGithub if needed
  ✓ Confirm before creating
  ✓ Create 9 labels
  ✓ Create 16 issues (13 epics + 3 tasks)
  ✓ Show you the URL to view them

Time needed: ~60 seconds

Problem: "Permission denied"
Solution: chmod +x quick-start.sh

Problem: "Python not found"
Solution: Install Python 3: https://python.org/downloads

Problem: "PyGithub not found"
Solution: pip install PyGithub

Problem: "401 Bad credentials"
Solution: Check your token has 'repo' scope
